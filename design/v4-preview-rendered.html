Quickstart

docToolchain follows the docs-as-code philosophy: your documentation lives next to your source code, written in plain text and built by a reproducible toolchain.

Prerequisites

The wrapper manages Java and dependencies. You only need one execution environment such as local, docker or sdk.

ℹ️docToolchain requires Java 17. The local environment can install it for you.

Choosing an environment

Most teams start with the local environment.

Advanced: Docker

Run everything inside a container for full reproducibility.

Proxy configuration

Set credentials via dtcw_docker.env.

Environment variables

All six heading levels are colour-differentiated.

Generate the site

Place AsciiDoc sources under src/docs, then run a single command:

./dtcw local generateSite
