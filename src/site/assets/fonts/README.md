# Bundled webfonts — docToolchain v4 microsite

These fonts are **self-hosted** (served from this site, not from a third-party
CDN) so no visitor IP is sent to Google Fonts (Quality Goal #1: no implicit
cloud). All three families are licensed under the **SIL Open Font License 1.1
(OFL)**, which explicitly permits bundling, redistribution and embedding. The
full licence text and copyright for each family is in the `OFL-*.txt` files in
this directory — included as the OFL requires.

| Family | Copyright | Licence |
|--------|-----------|---------|
| Space Grotesk | Copyright 2020 The Space Grotesk Project Authors | `OFL-Space-Grotesk.txt` |
| IBM Plex Sans | Copyright 2017 IBM Corp. — Reserved Font Name "Plex" | `OFL-IBM-Plex-Sans.txt` |
| IBM Plex Mono | Copyright 2017 IBM Corp. — Reserved Font Name "Plex" | `OFL-IBM-Plex-Mono.txt` |

The `*.woff2` files (latin subset) were sourced from [fontsource](https://fontsource.org).
The fonts are **unmodified** and keep their original names — no Reserved Font Name
is used for a modified version, so the OFL's Reserved-Font-Name condition is met.
